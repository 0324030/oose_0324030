# 系統內容：

建立一個媒合平臺，包含產品、工人、設計師與廠商。

(1)客戶購賣產品。

(2)客戶購賣產品，請工人安裝、組裝。

(3)客戶購賣產品，請設計師幫忙室內設計。

(4)客戶找不到想要的產品，請設計師設計。

(5)客戶有產品設計圖，請廠商生產。

**客戶可以是任何人或角色。**
